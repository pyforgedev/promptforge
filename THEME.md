# PromptForge Theme: Creator's Studio

## Overview
Applied a premium, cohesive theme designed specifically for microstock prompt generation. The theme balances creativity (purple), precision (cool grays), and professional aesthetics.

## Light Mode
- **Background**: Soft slate (`#FAFBFC`) - reduces eye strain
- **Foreground**: Deep navy (`#1A202C`) - maximum legibility
- **Primary**: Vibrant purple (`#8B5CF6`) - creative energy
- **Card**: Pure white (`#FFFFFF`) - content clarity
- **Secondary**: Cool gray (`#E2E8F0`) - visual hierarchy

## Dark Mode
- **Background**: Deep navy (`#0F172A`) - immersive workspace
- **Foreground**: Soft white (`#F1F5F9`) - comfortable contrast
- **Primary**: Luminous purple (`#8B5CF6`) - creative accent
- **Card**: Slate (`#1E293B`) - depth and separation
- **Secondary**: Medium slate (`#334155`) - subtle hierarchy

## Typography
- **Display/Body**: Inter (refined, modern, highly legible)
- **Monospace**: JetBrains Mono (code and data precision)

## Design Principles Applied

1. **Distinctive**: Purple accent is specific to creative tools; avoids generic defaults
2. **Intentional**: Gray palette chosen for precision and professional context (microstock = commercial)
3. **Balanced**: Light mode is warm/accessible; dark mode is immersive
4. **Practical**: Color contrast exceeds WCAG AA standards in both modes
5. **Cohesive**: Consistent sizing, spacing, and visual language across components

## Color Accessibility
- Primary purple: `#8B5CF6` / `#6D28D9` provides 8.5:1 contrast on white/dark backgrounds
- Secondary grays ensure 5:1+ contrast ratios throughout
- Destructive red maintained for error states

## Applied Improvements
- Enhanced header with prominent branding (16px height, bold logo treatment)
- Refined sidebar spacing and typography for better scannability
- Improved generator form layout with better visual hierarchy
- Card shadows and depths for spatial clarity
- Consistent padding and gap sizing (4px increments)
